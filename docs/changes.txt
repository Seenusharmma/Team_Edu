MOBILE APP CHANGES:
-------------------

1. STICKY NAVBAR (HeroSection.tsx)
   - Created separate Navbar component
   - Fixed position at top using absolute positioning
   - Modern glassmorphism design with rounded corners
   - Logo with "S" icon and "Siksha" brand name
   - Contact Us button navigates to /contact
   - Animated entrance (fade + scale)
   - z-50 to stay on top of content
   - Safe area padding for notch devices

2. FEATURES SECTION CAROUSEL
   - Auto-scrolling carousel (6 second intervals)
   - Manual swipe navigation with pan gesture
   - Dot pagination with animated active state
   - Same card design as TestimonialsSection
   - Infinite scrolling behavior
   - Smooth snap animations (600ms duration)
   - Velocity-based swipe detection
   - Cards use white background with border and shadow

3. BOTTOM TAB BAR (_layout.tsx)
   - Premium Ionicons from @expo/vector-icons
   - Active/Inactive icon states (filled vs outline)
   - Shadow and border styling
   - 70px height with proper padding
   - Modern glass-like appearance
   - Icons: home, book, bar-chart, person

4. CONTACT PAGE (contact.tsx)
   - Full contact form with name, email, subject, message
   - Modern input styling with rounded corners
   - Send Message button with accent color
   - Get in Touch section with contact info
   - Keyboard avoiding behavior
   - Back navigation button

5. GESTURE HANDLER SETUP
   - Added GestureHandlerRootView to root _layout.tsx
   - Required for GestureDetector and pan gestures

6. VIDEO PLAYER FIX
   - Fixed allowsFullscreen deprecation
   - Changed to fullscreenOptions={{ enable: false }}
